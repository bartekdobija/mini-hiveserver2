# mini-hiveserver2
Mini HiveServer2 to test HiveQL scripts.

Maven configuration:

```xml
<repository>
  <id>jitpack.io</id>
  <url>https://jitpack.io</url>
</repository>

<dependency>
  <groupId>com.github.bartekdobija</groupId>
  <artifactId>mini-hiveserver2</artifactId>
  <version>v0.7</version>
</dependency>
```
